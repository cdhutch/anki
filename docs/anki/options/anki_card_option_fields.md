# Anki Card Option Fields

## Daily Limits
- New cards/day
- Max reviews/day
- New cards ignore review limit (Y/N)
- Limits start from top (Y/N)

## New Cards
- Learning Steps
- Insertion order

## Lapses
- Relearning steps
- Leech threshold
- Leech action

## Display Order
- New card gather order
- New card sort order
- New/review order
- Interday learning/review order
- Review sort order

## FSRS
- Enabled (Y/N)
- Desired Retention (Preset or This Deck; value)
- Reschedule cards on change (Y/N)
- Check health when optimizing (Y/N)

## Burying
- Bury new siblings (Y/N)
- Bury review siblings (Y/N)
- Bury interday learning siblings (Y/N)

## Timers
- Maximum answer seconds
- Show on-screen timer (Y/N)
- Stop on-screen timer on answer (Y/N)

## Auto Advance
- Seconds to show question for
- Seconds to show answer for
- Question action
- Answer action

## Advanced
- Maximum interval
- Historical retention
