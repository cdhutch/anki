# front_md

**ENGINES — IGNITION SELECT SWITCH**

What are the positions of the **Ignition Select Switch** and what does each position provide?

# back_md

The **Ignition Select Switch** has **three positions**:

- **IGN L**
- **BOTH**
- **IGN R**

---

### IGN L

- Provides ignition to the **left igniter only** for **both engines**.

### BOTH (normal position)

- Provides ignition to **both igniters** for **both engines**.

### IGN R

- Provides ignition to the **right igniter only** for **both engines**.

---

The switch allows the crew to **alternate igniter usage** between engine starts or select **both igniters when desired**.
