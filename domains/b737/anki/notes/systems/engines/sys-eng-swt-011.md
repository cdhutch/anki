# front_md

**ENGINES — START SWITCH POSITIONS**

What are the engine start switch positions and their functions?

# back_md

Engine start switch positions:

- **GRD**
  - Opens the start valve
  - Allows pneumatic air to drive the starter

- **OFF**
  - Closes the start valve

- **CONT**
  - Provides continuous ignition

- **FLT**
  - Provides ignition for in-flight start or severe conditions
