# front_md

**ENGINES — DISPLAY CONTROL PANEL**

What is the function of the Display Control Panel for engine indications?

# back_md

The display control panel allows pilots to control the presentation of engine data on the flight deck displays.

It allows:

- Selection of display formats
- Selection of display sources
- Control of brightness and display configuration.
