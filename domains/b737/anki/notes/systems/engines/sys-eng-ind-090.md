# front_md

**ENGINES — EGT START VALUES**

What EGT values are normally seen during engine start?

# back_md

Typical EGT indications during start:

737-800 (CFM56-7B)

- Normal start peak: **≈ 500–600°C**
- Idle EGT after start: **≈ 400–450°C**

737 MAX-8 (LEAP-1B)

- Normal start peak: **≈ 450–550°C**
- Idle EGT after start: **≈ 350–420°C**

Abnormally high EGT during start may indicate:

- hot start
- insufficient starter airflow
- improper fuel introduction timing
