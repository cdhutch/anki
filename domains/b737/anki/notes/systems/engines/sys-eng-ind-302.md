# front_md

**ENGINES — SECONDARY ENGINE PARAMETERS**

When are secondary engine parameters automatically displayed?

# back_md

Secondary engine parameters automatically appear when:

- **An engine fails**
- **An engine parameter exceeds its limit**

Secondary parameters include:

- **N2**
- **Oil pressure**
- **Oil temperature**
- **Engine vibration**
