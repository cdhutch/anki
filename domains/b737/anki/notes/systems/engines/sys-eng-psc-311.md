# front_md

**ENGINES — ENGINE PANEL (MAX-8)**

What controls and indications are located on the **MAX-8 Engine Panel**?

# back_md

The **MAX-8 Engine Panel** includes:

- **EEC switches**
- **REVERSER LIMITED lights**
- **REVERSER COMMAND light**
- **REVERSER AIR/GRD light**
- **ENGINE CONTROL lights**

These controls provide monitoring and control of:

- **LEAP-1B Electronic Engine Control**
- **Thrust reverser system status**
- **Engine control system faults**
