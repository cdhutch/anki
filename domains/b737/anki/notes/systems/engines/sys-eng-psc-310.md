# front_md

**ENGINES — ENGINE PANEL (-800)**

What controls and indications are located on the **737-800 Engine Panel**?

# back_md

The **737-800 Engine Panel** includes:

- **EEC switches**
- **REVERSER lights**
- **ENGINE CONTROL lights**

These components allow the crew to:

- Control **Electronic Engine Control operation**
- Monitor **thrust reverser status**
- Monitor **engine control system faults**
