# front_md

**Go-Around — Single Engine — PM — Initial Actions**

What are the PM actions from initiation of a single-engine go-around through gear up and missed approach execution?

# back_md

**Go-Around — Single Engine — PM — Initial Actions**

- After PF callout "Go around, TOGA.":
  - Verify go-around thrust
- After PF command "Flaps 1.":
  - Select flaps 1
  - Verify FMA
  - Verify thrust
- Positive rate of climb:
  - "Positive rate."
- After PF command "Gear up.":
  - Select gear up
- After PF command "Set missed approach altitude.":
  - Set missed approach altitude
  - Advise ATC
