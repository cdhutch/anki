# front_md

**Engine-Out SID (EO SID) Missed Approach**

What are the steps to select the EO SID during a single-engine missed approach?

# back_md

**Engine-Out SID (EO SID) Missed Approach**

Execute before reaching the first EO SID waypoint (may be required before 400' AGL):

1. LEGS Key (if LEGS not already displayed) — Press
2. RTE 2 LEGS Key [6L] — Press
3. First EO SID waypoint [1L] — Enter
4. ACTIVATE — Select
5. EXEC Key — Press
6. LNAV — Select or verify engaged

*Note: HDG SEL may be used to track EO SID lateral path if required.*  
*Note: Open speed window to comply with published speed/configuration requirements if required.*  
*Caution: Terrain and obstacle clearance only assured when following the published EO SID lateral path.*
