# back_md

**Rejected Takeoff — First Officer — Initial Actions**

If aircraft control is transferred:

- "Your aircraft."

Verify and monitor:

- Thrust levers closed
- Autothrottles disengaged
- Maximum braking applied

Call out:

- Any omissions
- “Autobrakes off” if AUTO BRAKE DISARM illuminates
- “Deployed” when speedbrake deploys
- “No Speedbrakes” if not deployed / not maintained
- “Single reverse” or “No reverse” if REV indications abnormal

Monitor deceleration throughout reject.

**Memory rule:**  
*Verify • Monitor • Call deviations*
