What are the PM actions for a PRM climbing breakout?

# back_md

**PRM Climbing Breakout — PM**

Trigger: ATC advises "Traffic alert"

- If on approach with LOC/GS captured:
  - Select both Flight Directors **OFF**, then **ON**
- Set desired heading on MCP
- Select **HDG SEL**
- Set desired altitude on MCP
- Select appropriate pitch mode (LVL CHG or V/S)
- Verify all needed actions have been completed
- Respond to "Set speed ___." callout:
  - Set speed as requested
  - Monitor flight path and speed; call out deviations
- Leveled off and established on heading:
  - Reconfigure aircraft as desired
