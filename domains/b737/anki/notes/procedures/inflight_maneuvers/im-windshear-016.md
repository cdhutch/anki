# front_md

## Takeoff Precautions (By Fleet)

What takeoff windshear precautions are fleet-specific for the NG -800 and MAX?

# back_md

## NG -800 (3AA-3PX)

- Use maximum **26K thrust** or **27K** if planned
- Use **flaps 5/15**

## MAX (3RA-3VM)

- Use **TO thrust**
- Use **flaps 5/10/15**

## Memory Items

NG -800 = max planned thrust, flaps 5/15; MAX = TO thrust, flaps 5/10/15
