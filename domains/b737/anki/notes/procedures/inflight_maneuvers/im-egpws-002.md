# front_md

What key words in an EGPWS aural alert indicate an immediate warning-response maneuver?

# back_md

The presence of the words **PULL UP**.

Examples:

- PULL UP
- TERRAIN TERRAIN PULL UP
- OBSTACLE OBSTACLE PULL UP

If the alert includes **PULL UP**, treat it as an immediate terrain warning requiring escape maneuver action.
