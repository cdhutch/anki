# front_md

How should the crew respond to an EGPWS alert that does **not** contain the words **PULL UP**?

# back_md

Treat it as a **caution** alert unless conditions clearly indicate otherwise.

Correct the:

- flight path
- aircraft configuration
- airspeed

If on approach and stabilized approach criteria cannot be met, execute a **go-around**.

Key distinction: **No PULL UP = correct the problem.**
