# front_md

How is an **RNAV Visual** approach identified on the chart and in the FMS?

# back_md

- **Chart name:** includes "RNAV VISUAL" (e.g., *KMCO RNAV VISUAL Rwy 17L*)
- **FMS name:** includes RNVV + runway (e.g., *RNVV17L*)
