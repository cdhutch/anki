# front_md

How do you know if an **FMS/CVFP** has associated RNAV waypoints in the FMS database, and how is it identified on the chart and in the FMS?

# back_md

Not all CVFPs have an associated FMS procedure.

If it **does** have RNAV waypoints in the database:

- **Chart name:** includes **(RNV)** in the title (e.g., *FMS BRIDGE VISUAL (RNV-F) Rwy 28R*)
- **FMS name:** includes the approach type the CVFP is based on + runway (e.g., *RNV28R-F*, *ILS28L-H*)
