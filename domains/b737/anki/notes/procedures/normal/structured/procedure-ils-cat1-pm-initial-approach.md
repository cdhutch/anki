# front_md

**ILS CAT I — PM — Initial Approach**

What are the PM actions from cleared for approach through glideslope capture on an ILS CAT I?

# back_md

**ILS CAT I — PM — Initial Approach**

- Arm APP, if requested
- After "LOC capture":
  - Verify VOR/LOC on FMA
- After "Gear down":
  - Select gear down
  - Set speed, if requested
- After "Flaps 15":
  - "Speed verified."
  - Set flaps 15
  - Set speed, if requested
- After "Glideslope capture. Set missed approach altitude":
  - Set missed approach altitude, if requested
- At final approach fix:
  - Verify Final Approach Verification Altitude
