# front_md

**ILS CAT II/III — First Officer (PM) — Final Configuration**

What are the First Officer (PM) actions from the Final Approach Verification Altitude Fix through final landing configuration on an ILS CAT II/III?

# back_md

**ILS CAT II/III — First Officer (PM) — Final Configuration**

- Configured by 1000 AGL:
  - Dual engine:
    - "Speed verified."
    - Set requested flaps
    - set speed, if requested
    - Complete Below the Line checklist
  - Engine out:
    - Complete Below the Line checklist
- Confirm mechanical checklist complete
