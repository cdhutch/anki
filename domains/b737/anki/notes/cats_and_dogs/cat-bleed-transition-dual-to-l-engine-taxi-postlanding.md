# front_md

**Bleed/Pack/ISO — Transition**

After landing: Taxiing on both engines → Taxiing on L engine: what switch changes are made?

# back_md

**Changes:**

| Switch        | From    | To           |
| --------      | ------  | ----         |
| Engine Bleeds | Both ON | L OFF / R ON |
| APU Bleed     | OFF     | ON           |

**New Configuration:**

| Switch        | Position     |
| --------      | ----------   |
| ISO Valve     | AUTO         |
| Packs         | Both AUTO    |
| Engine Bleeds | L OFF / R ON |
| APU Bleed     | ON           |
