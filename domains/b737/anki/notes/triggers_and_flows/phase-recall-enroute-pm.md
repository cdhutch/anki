# front_md

**Enroute — Pilot Monitoring**

What does the PM do during Enroute (Cruise and Prior to TOD)?

# back_md

**Enroute — PM** *(Cold Jet → Move → Roll → Fly → Cruise → **Down** → Land → Park)*

**PM Cruise** — mnemonic: **FLARE**

- **F** — Fuel: Center Fuel Pumps
- **L** — Log: Fuel Log
- **A** — Announce: Initial Climb / Cruise PA
- **R** — Report: Company Position Report
- **E** — Escape: Depressurization Routes

**PM Prior to TOD** — mnemonic: **MAAP TRAC**

- **M** — Mech: Mechanical Discrepancy Notifications
- **A** — ATIS
- **A** — Approach Preparation
- **P** — Pressurization
- **T** — Terrain Display
- **R** — Recall Check
- **A** — Announce: Before Descent Announcement
- **C** — Changeover Report
