**Engine Start Procedure**

1. PACKS: OFF
2. ENGINE START switch: GND
3. At 25% N2, or max motoring > 20% N2 with N1 rotation:
   - If 737-800: START LEVER: IDLE
   - If MAX 8: After MOTORING indication blanks, START LEVER: IDLE
4. At 56% N2 (737-800) or 63% N2 (MAX 8), verify:
   - ENGINE START switch moves to AUTO; if not, position switch to AUTO
   - START VALVE OPEN alert extinguishes within 5 seconds of ENGINE START switch to AUTO
   - Increase in oil pressure by the time engine stabilizes at idle
