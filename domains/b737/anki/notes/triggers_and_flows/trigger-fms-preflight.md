**FMS Preflight**

- Complete FMS preflight prior to the Before Start Flow
- CDU preflight procedure may be started anytime prior to General Preflight Flow

Initialization timing:

- Initial navigation data entries: before General Preflight Flow flight instrument check
- Navigation and performance data entries: prior to Before Start Checklist
