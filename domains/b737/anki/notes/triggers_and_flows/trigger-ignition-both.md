# front_md

**Ignition — BOTH**

When should one consider setting the ignition switch set to BOTH for engine start?

# back_md

**Ignition — BOTH**

Consider BOTH when **all three** apply:

- First flight of the day
- Airport elevation **≥ 2000' MSL**
- OAT **< 5°C / 41°F**
