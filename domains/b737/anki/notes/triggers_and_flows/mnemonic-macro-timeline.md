# front_md

**B737 Macro Flight Timeline**

What are the 9 phases of flight, and what is the mnemonic?

# back_md

Mnemonic: **Cold Jet → Move → Roll → Fly → Cruise → Down → Land → Park**

| Mnemonic word | Phase            |
| ---           | ---              |
| **Cold Jet**  | Preflight        |
| **Move**      | Pushback / Start |
| **Roll**      | Taxi             |
| **Fly**       | Takeoff          |
| **Cruise**    | Departure        |
| **Down**      | Enroute          |
| **Land**      | Arrival          |
| **Park**      | After Landing    |
| *(then)*      | Shutdown         |

*Note: 8 words cover 8 phases; Shutdown follows naturally after Park.*
