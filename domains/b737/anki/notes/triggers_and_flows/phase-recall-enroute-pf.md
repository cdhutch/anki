# front_md

**Enroute — Pilot Flying**

What does the PF do during Enroute (Cruise and Prior to TOD)?

# back_md

**Enroute — PF** *(Cold Jet → Move → Roll → Fly → Cruise → **Down** → Land → Park)*

**PF Cruise** — mnemonic: **SWEFT**

- **S** — Speed: Standard Descent Speed
- **W** — Winds: Update Cruise Winds
- **E** — Enter: Enter Descent Forecast
- **F** — FMS+
- **T** — Trim: Minimum Drag Trim Setting

**PF Prior to TOD** — mnemonic: **LAATE** *(do this before it's too* late*)*

- **L** — Load Arrival in FMS
- **A** — Approach Preparation
- **A** — Autobrakes
- **T** — Terrain Display
- **E** — Engine Anti-Ice
