# front_md

**B737 LIMITS — TAKEOFF CROSSWIND (COMMON CONDITIONS)**

Recall the common takeoff crosswind limits by runway condition:

# back_md

| Condition                       | Limit (kt) |
| ---                             | ---:       |
| Wet                             | 25         |
| Standing water / slush > 1/8 in | 15         |
| Snow (wet and dry)              | 25         |
| Compacted snow                  | 20         |
| Ice                             | 15         |
| Narrow runway (< 148 ft / 45 m) | 20         |
