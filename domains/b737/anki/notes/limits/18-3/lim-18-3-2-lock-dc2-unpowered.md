# front_md

**B737 LIMITS — FLIGHTDECK DOOR**

How can the **flightdeck door be locked if DC Bus 2 is unpowered**?

# back_md

1. Place the **POWER CUTOFF switch** to **OFF**

2. Use the **deadbolt** in the: *LOCKED - KEY INOPERABLE* position.
